## 2. 产品定义

### 2.1 产品愿景

让个人用户在任何设备上都能拥有一个**完全自主可控、离线可用**的番茄钟工作流，
从专注计时到任务管理再到数据回顾，形成完整的个人效能闭环。

### 2.2 三个清晰目标

1. **可靠的专注伴侣** — 提供稳定、流畅的番茄钟计时体验，支持 PWA 离线运行，
   断网不中断，联网自动同步。音效和通知确保用户不会错过休息/工作切换。

2. **任务驱动的效率闭环** — 番茄钟不再只是倒计时，而是与任务深度绑定：
   每个番茄关联一个任务，完成任务时看到累计的专注投入，让时间花得明明白白。

3. **自主可控的数据洞察** — 所有数据存储在用户自己的机器上，不为任何 SaaS
   服务付费或让渡隐私。通过统计图表和反思记录，用户能全景式回顾自己的专注习惯。

### 2.3 用户故事

| # | 用户故事 | 验收标准 |
|---|----------|---------|
| US1 | **作为用户，我希望打开浏览器就能开始一个番茄钟，选择专注/短休/长休模式，计时结束后自动切换。** | 计时器页面路由 `/timer` 为首页；支持 work / short_break / long_break / free 四种模式；倒计时结束后播放音效并弹出桌面通知；自动提议下一段模式（work→break）。 |
| US2 | **作为用户，我希望创建任务并关联到番茄钟，每个任务有预估番茄数和实际完成数。** | 任务列表路由 `/tasks`；支持列表/看板/日历三视图；任务字段包括标题、描述、状态(todo/in_progress/done/archived)、优先级(low/medium/high/urgent)、标签、预估番茄数、实际番茄数；进行番茄时可选择绑定任务。 |
| US3 | **作为用户，我希望查看每日/每周的专注数据，包括总时长、完成番茄数、心情分布，帮助我了解自己的效率趋势。** | 统计页路由 `/stats`；支持日期范围选择；图表展示番茄数趋势、时长分布、心情饼图；数据支持按任务和按日期聚合。 |
| US4 | **作为用户，我希望番茄钟结束时能快速写一段反思笔记，记录当前心情和想法，并能按心情标签筛选回顾。** | 番茄结束时弹出反思编辑器；心情用 5 级表情记录（great/good/normal/bad/terrible）；支持 Markdown 编辑和预览；反思页路由 `/reflections`；支持心情筛选和关键词搜索。 |
| US5 | **作为用户，我希望在没有网络的环境下（如飞机、地铁）正常使用番茄钟和任务管理，恢复网络后数据自动同步。** | PWA 可安装到桌面/主屏；离线时使用 IndexedDB 本地存储；联网后通过 outbox 事件流增量同步；不丢数据不重复。 |

## 6. 数据模型概要

> 详细模型定义见 `docs/architecture.md` Section 8.9，此处仅列 PRD 需关注的业务字段。

### Task（任务）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| title | string | 任务标题 |
| description | string | 任务描述（可选） |
| status | enum | `todo` / `in_progress` / `done` / `archived` |
| priority | enum | `low` / `medium` / `high` / `urgent` |
| tags | string[] | 标签列表 |
| estimated_pomodoros | int | 预估番茄数，默认 1 |
| actual_pomodoros | int | 实际完成番茄数 |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |
| archived_at | datetime | 归档时间（null 表示未归档） |

### Session（番茄会话）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| task_id | int? | 关联任务（free 模式可为 null） |
| type | enum | `work` / `short_break` / `long_break` / `free` |
| started_at | datetime | 开始时间 |
| ended_at | datetime? | 结束时间 |
| duration | int | 时长（秒） |
| completed | boolean | 是否完成（非中断） |
| plan | string? | 计划笔记（开始前写） |
| completion | string? | 完成笔记（结束后写） |
| updated_at | datetime | 更新时间 |

### Reflection（反思）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| content | string | 反思内容 (Markdown) |
| mood | enum | `great` / `good` / `normal` / `bad` / `terrible` |
| tags | string[] | 标签 |
| related_task_ids | int[] | 关联任务 ID |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |
