## 一、项目结构

```
PomodoroXII/
├── backend/                 #  canonical 应用（FastAPI）
│   ├── app/                 #  ~110 Python 模块
│   ├── alembic/versions/    #  7  migrations
│   ├── tests/               #  64+ 测试文件
│   ├── scripts/             #  scaffold_entity, check_entity_consistency
│   ├── pyproject.toml, Dockerfile
│   └── .codebase-memory/    #  CBM 图谱 artifact
├── .github/workflows/ci.yml
├── 核心文档/                #  15 份架构规划
├── documents/               #  v4 开发规划等
├── 审计报告/
├── .trae/documents/         #  41+ 执行计划
└── overview.md, 深度审查报告-独立复核版.md
```

**非 monorepo：** 无 `frontend/`、`package.json`、`docker-compose`（注：下午 main 已新增 compose）。

**架构：** 共享 FastAPI + meta DB（`meta.db`）+ 每空间 SQLite + 每空间文件系统（`.md` + `index.db` FTS5）。

## 三、已实现能力

### 3.1 架构亮点

- **三条铁律 100% 合规：** services 层零 `commit`，32 处 commit 全在 routes
- **Sync 5 道安全检查：** 墓碑防复活、字段剥离、文件夹循环引用、SAVEPOINT、批量审计
- **多空间 LRU 引擎池：** 双重检查锁 + 异步 dispose
- **NoteService Saga：** FS `.md` 为 SoT，DB 存 hash/word_count，双写补偿
- **RelationService：** unlink 写 junction tombstone
- **YAML frontmatter：** `file_system/frontmatter.py` + 测试（旧审计称未实现，**代码已实现**）

### 3.2 REST API（`/api/v1`）

|--------|------|
| auth / spaces / meta | ✅ 完整（master token） |
| tasks / folders / notes / quick-notes / trash / stats / settings / sync | ✅ 完整 |
| sessions / reflections / schedules / time-blocks | ⚠️ 无 PUT |
| habits + check-ins | ⚠️ habit 无 PUT |
| search / export / consistency / relations | ❌ FS 层有，HTTP 未暴露 |
