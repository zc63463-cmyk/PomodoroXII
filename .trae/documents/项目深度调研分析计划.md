待执行的 Cypher 查询清单（在执行阶段批量运行）:

```cypher
// Q1: 高复杂度函数（>5）— 找出需要重构的热点
MATCH (f:Function) WHERE f.complexity > 5
RETURN f.qualified_name, f.complexity, f.cognitive, f.loop_depth
ORDER BY f.complexity DESC LIMIT 20

// Q2: 高扇入节点（被广泛调用的核心函数）
MATCH (n) WHERE n.fan_in > 10
RETURN n.qualified_name, n.fan_in
ORDER BY n.fan_in DESC LIMIT 15

// Q3: 跨边界调用（违反三层铁律的嫌疑）
MATCH (r:Function)-[:CALLS]->(s:Function)
WHERE r.file_path STARTS WITH "app/routes" AND s.file_path STARTS WITH "app/models"
RETURN r.qualified_name, s.qualified_name

// Q4: 服务层是否泄露 fastapi import（应 0 个）
MATCH (m:Module)-[:IMPORTS]->(target)
WHERE m.qualified_name STARTS WITH "app.services" AND target CONTAINS "fastapi"
RETURN m.qualified_name, target

// Q5: 测试覆盖缺口（每个 service 函数的 TESTS 边数）
MATCH (f:Function)-[:TESTS]->(target:Function)
WHERE f.file_path STARTS WITH "tests/" AND target.file_path STARTS WITH "app/services"
RETURN target.qualified_name, count(f) AS test_count
ORDER BY test_count ASC
```
