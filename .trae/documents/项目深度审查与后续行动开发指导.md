### P0 — 数据一致性风险（阻塞 Phase C）

#### P0-1: DB 表隔离未实现

**位置**:
- [meta_session.py:46](file:///e:/Development/MyAwesomeApp/PomodoroXII/backend/app/db/meta_session.py#L46) — `Base.metadata.create_all` 无 `tables=` 参数
- [space_manager.py:152](file:///e:/Development/MyAwesomeApp/PomodoroXII/backend/app/space_manager.py#L152) — `Base.metadata.create_all` 无排除 meta 表
- [alembic/env.py:18](file:///e:/Development/MyAwesomeApp/PomodoroXII/backend/alembic/env.py#L18) — 无 `include_object` 过滤器

**影响**: meta DB 可能被创建 18 张业务表；每个 space DB 多出 2 张 meta 表。`conftest.py:93` 注释 "all 20 tables" 证实此问题已存在。

**修复方案**:
```python
# meta_session.py — init_meta_db() 只创建 meta 表
from app.db.models.meta import Space, MetaSetting
await conn.run_sync(Base.metadata.create_all, tables=[Space.__table__, MetaSetting.__table__])

# space_manager.py — _init_schema() 排除 meta 表
_meta_tables = {Space.__table__, MetaSetting.__table__}
space_tables = [t for t in Base.metadata.sorted_tables if t not in _meta_tables]
await conn.run_sync(Base.metadata.create_all, tables=space_tables)

# alembic/env.py — 添加 include_object 过滤器
def include_object(object, name, type_, reflected, compare_to):
    target = config.attributes.get("target", "space")
    if target == "meta":
        return name in ("spaces", "meta_settings")
    return name not in ("spaces", "meta_settings")
```

**新增测试**: `tests/test_db_isolation.py`（3 个测试）
- `test_meta_db_has_only_2_tables` — meta DB 只含 spaces + meta_settings
- `test_space_db_excludes_meta_tables` — space DB 不含 spaces / meta_settings
- `test_space_db_has_all_business_tables` — space DB 含 18 张业务表

**位置**: [note.py](file:///e:/Development/MyAwesomeApp/PomodoroXII/backend/app/services/note.py)

| 方法 | 行号 | 问题 | 补偿方案 |
|------|------|------|----------|
| `create()` | L52-67 | FS 写入成功后 DB 失败 → 孤儿 .md 文件 | DB 失败时调用 `fs.delete_note(meta.id)` 回滚 |
| `update_content()` | L75-82 | FS 编辑成功后 DB 失败 → content_hash 不一致 | 保存 old_content，DB 失败时 `fs.edit_note(id, old_content)` 恢复 |

**delete() 顺序修正**:
```python
# 当前（错误）：FS → DB → tombstone
# 修正为：DB delete + tombstone → FS best-effort
async def delete(self, id: str) -> None:
    obj = await self.db.get(self.model, id)
    if obj is not None:
        await self.db.delete(obj)
        await self.db.flush()
    await TombstoneService(self.db).create("note", id)
    try:
        await self.fs.delete_note(id)
    except (KeyError, FileNotFoundError):
        pass
```
