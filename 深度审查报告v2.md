## 六、安全审查

### 6.1 安全亮点 ✅

- PyJWT + bcrypt 12 rounds + 算法固定 `algorithms=[settings.algorithm]`
- 双 token 模型（master 7d / space 8h）+ space_id 实时校验防伪造
- 多空间隔离通过独立 SQLite DB 实现
- FTS5 双引号转义防注入（search_ops.py:40）
- 参数化查询贯穿全栈
- 500 错误不泄露堆栈
- 关联表 link 用 IntegrityError 捕获处理 TOCTOU（relation.py:65-77）
- .env 未被 git 跟踪

### 6.2 安全缺口

| 项 | 严重度 | 说明 |
|----|--------|------|
| 无认证限流 | MAJOR | /auth/login 无 IP 级速率限制 |
| 无安全响应头 | MAJOR | 缺 X-Content-Type-Options/X-Frame-Options/HSTS/CSP |
| HMAC 密钥 29 字节 | MINOR | 低于 SHA256 推荐的 32 字节（测试 warning 印证） |
| CORS 开发环境较宽 | MINOR | allow_credentials=True + 显式列表（非通配，正确） |
